# Rivus Frontend

Landing page for the Rivus payment streaming protocol. Built with Next.js 16 and Tailwind CSS.

## Local development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
```

## Deploy

Deployed on Vercel. Set **Root Directory** to `./frontend` in the Vercel dashboard.
